2. Describe the purpose of a DIV tag.

The purpose of a DIV tag is to tell the difference between one element of code to another. It also allows us to use specific code for only that one div on CSS.

10. Why do we use media queries?

We use media queries because media queries allows us to have responsive design and helps us arrange our webpages to adjust to multiple platforms like desktops, tablets. and mobile phones.

11. What is the main difference between a serif and a sans-serif font?

A serif font has small line at the end of each letter and often looks more elegant than sans serif fonts but serif fonts are mainly used for headlines. Sans serif fonts don't have the small line but are used on web because it's easier to read them.
